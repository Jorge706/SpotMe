# SpotMe — iOS Client

Native iOS client for the SpotMe tracking platform, written in **Swift**.

## Features

- **Live tracking on the go** — follow vehicle positions and trip activity from an iPhone.
- **Secure sessions** — authenticates against the Users API (JWT), including password management flows.
- **Native UIKit interface** — view-controller-based architecture.

## Stack

Swift · UIKit · Xcode

## Running locally

Open `SpotMe/SpotMe.xcodeproj` in Xcode, set your development team, and run on a simulator or device.

Part of the [SpotMe monorepo](https://github.com/Jorge706/SpotMe) — see the root README for the full architecture.
